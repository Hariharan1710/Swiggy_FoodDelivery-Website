Swiggy Food Ordering Webpage (UI Clone)

This project is a front-end recreation of the Swiggy food ordering platform, designed to replicate its clean layout, modern UI components, and user-friendly structure. The webpage showcases a responsive and visually appealing interface that highlights essential features commonly found in food delivery applications.

Key Features
1. Header Section

A Swiggy-style logo placed on the top-left.

A search bar allowing users to look for restaurants or dishes.

Navigation links including Offers, Help, Sign In, and a Bag/Cart icon.

2. “What’s on your mind?” Section

A horizontally scrollable menu with food categories such as:

Biryani

Noodles

South Indian

Pizzas

Cakes

Desserts

Each category is represented with a clean circular image and label.

3. Top Restaurant Chains in India

A featured section displaying some well-known restaurant chains.

Each card contains:

Restaurant image

Restaurant name

Rating

Type of food offered (e.g., snacks, desserts, South Indian)

Cards are designed with rounded edges and a clean aesthetic, similar to Swiggy's original UI.

Technologies Used

HTML5 for structure

CSS3 for layout, styling, and spacing

Optional: JavaScript for additional interactivity (if applied)

Purpose of the Project

To practice and demonstrate front-end web development skills.

To replicate a real-world UI for learning purposes.

Ideal for portfolio use, especially for students or beginners in web design.

Outcomes

The final webpage looks and feels like an actual food delivery platform, helping users understand professional UI/UX patterns while strengthening core HTML/CSS skills.
